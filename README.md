# CRAFT.pytorch
An unofficial pytorch implementation of CRAFT: Character Region Awareness for Text Detection.

## Overview
Part1:  train your model on the synthtext dataset.

Part2:  finetune your model on the ICDAR2015 dataset.

## Display
